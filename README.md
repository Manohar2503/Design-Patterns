# Design Patterns in Java

A comprehensive collection of commonly used **Design Patterns implemented in Java** to strengthen understanding of **Object-Oriented Programming (OOP)**, **Low-Level Design (LLD)**, and **Software Engineering principles**.

This repository demonstrates how design patterns help build **scalable, maintainable, reusable, and loosely coupled applications**.

---

## Features

- Implementations in **Java**
- Covers **Creational, Structural, and Behavioral Design Patterns**
- Follows **OOP concepts and SOLID principles**
- Clean and readable code
- Suitable for:
  - Software Engineer interviews
  - Low-Level Design (LLD) preparation
  - Java developers
  - Students learning software architecture

---

## Design Pattern Categories

### Creational Design Patterns
These patterns deal with object creation mechanisms.

- Singleton Pattern
- Factory Method Pattern
- Abstract Factory Pattern
- Builder Pattern
- Prototype Pattern

---

### Structural Design Patterns
These patterns focus on class and object composition.

- Adapter Pattern
- Bridge Pattern
- Composite Pattern
- Decorator Pattern
- Facade Pattern
- Flyweight Pattern
- Proxy Pattern

---

### Behavioral Design Patterns
These patterns define communication between objects.

- Strategy Pattern
- Observer Pattern
- State Pattern
- Command Pattern
- Template Method Pattern
- Chain of Responsibility Pattern
- Iterator Pattern
- Mediator Pattern
- Memento Pattern
- Visitor Pattern
